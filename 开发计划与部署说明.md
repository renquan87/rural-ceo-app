# "乡理人"点亮行动 — 开发计划与部署说明

## 一、项目概述

本项目是一个纯前端Web网页平台，面向大学生村官和返乡创业青年，提供乡村治理能力培训、成功案例学习、AI智能分析等功能。项目以展示为主，用于向评委演示平台的功能规划和交互体验。

**技术栈：** HTML + CSS + JavaScript（纯前端，无后端依赖）
**兼容性：** PC端浏览器为主，手机端自适应

## 三、需要你操作的部分

### 1. 部署到 GitHub Pages

**步骤：**

```bash
# 1. 在 GitHub 上创建一个新仓库，例如 "xiangliren"
#    访问 https://github.com/new 创建

# 2. 在本地初始化 Git 并推送
cd /data/projects/web-app/rural-ceo-app
git init
git add .
git commit -m "初始提交：乡理人点亮行动平台"

# 3. 关联远程仓库（替换为你的用户名和仓库名）
git remote add origin https://github.com/你的用户名/xiangliren.git
git branch -M main
git push -u origin main

# 4. 开启 GitHub Pages
#    - 进入仓库 Settings → Pages
#    - Source 选择 "Deploy from a branch"
#    - Branch 选择 "main"，文件夹选择 "/ (root)"
#    - 点击 Save
#    - 等待几分钟后，访问 https://你的用户名.github.io/xiangliren/
```

### 2. 自定义内容修改

如果需要修改数据内容（课程、案例、资料等），编辑 `js/data.js` 文件：

- **修改课程：** 找到 `coursesData` 数组，修改或添加课程对象
- **修改案例：** 找到 `casesData` 数组，修改或添加案例对象
- **修改资料：** 找到 `resourcesData` 数组
- **修改AI回复：** 找到 `aiResponses` 对象，修改各类别的回复模板
- **修改能力模块：** 找到 `abilityModules` 数组

### 3. 替换品牌图标（可选）

当前使用 Font Awesome 图标。如果需要自定义Logo：
1. 将Logo图片放入项目根目录
2. 在 `index.html` 中找到 `.brand` 部分，将 `<i class="fas fa-seedling"></i>` 替换为 `<img src="logo.png" alt="logo" style="height:28px">`

## 五、说明

- 本项目为纯前端演示项目，所有数据为静态模拟数据
- AI助手为模拟对话，根据关键词匹配预设回复，非真实AI接口
- 社区聊天为模拟交互，发送消息后会自动生成模拟回复
- 所有"收藏""开始学习"等操作为Toast提示演示，无实际后端存储
- 项目可直接用浏览器打开 `index.html` 运行，无需安装任何依赖
