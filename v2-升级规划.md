# "乡理人"点亮行动 — 网页美观升级规划 v2

## 一、当前问题分析

### 视觉层面
1. **色彩单调** — 主要依赖蓝紫渐变，缺乏层次感和视觉焦点变化
2. **卡片同质化** — 所有卡片样式雷同（白色毛玻璃+圆角），缺乏大小对比和视觉节奏
3. **图标占位** — 能力模块用单字（政/经/品）代替图标，案例/课程用纯色渐变代替封面图，显得廉价
4. **排版平淡** — 所有区块等宽等高排列，缺乏杂志式的排版层级和留白节奏
5. **动效不足** — 虽有基础hover和reveal，但缺乏让人"哇"的视觉冲击

### 结构层面
6. **首页信息密度低** — Hero卡片占据大量空间但信息量少
7. **页面间风格断裂** — 各页面独立渲染，缺乏统一的视觉叙事线
8. **空状态缺失** — 无数据时显示空白，缺乏友好引导

## 二、升级方案（按优先级排列）

### 🔥 P0 — 视觉冲击力（必做）

#### 2.1 Bento Grid 首页重构
将首页从"线性列表"改为"便当盒布局"，不同模块占据不同大小的格子：

```
┌──────────────┬──────────┐
│  Hero学习卡   │  AI助手   │
│  (2col,大)    │  快捷入口 │
├────┬────┬────┼──────────┤
│ 政 │ 经 │ 品 │ 今日任务  │
├────┼────┼────┤  (纵向)   │
│ 实 │ 营 │ 数 │          │
├────┴────┴────┼────┬─────┤
│  推荐案例(横向滚动)│排行│
└──────────────┴────┴─────┘
```

- 大卡片和小卡片交错，形成视觉节奏
- 统一圆角 20px，间距 16px
- 每个格子有独立的渐变背景色

#### 2.2 真实图片替换
**这是提升质感最关键的一步。** 用真实照片替换所有纯色渐变占位：

| 位置 | 数量 | 建议内容 | 文件命名 |
|------|------|----------|----------|
| 案例封面 | 8张 | 各地乡村实景照 | `images/case-1.jpg` ~ `case-8.jpg` |
| 课程缩略图 | 6张 | 与课程主题相关 | `images/course-1.jpg` ~ `course-6.jpg` |
| Hero背景 | 1张 | 乡村全景/航拍 | `images/hero-bg.jpg` |
| 能力模块图标 | 6张 | SVG插画图标 | 可用在线图标库替代 |

> 图片来源建议：Unsplash搜索 "chinese village"、"rural china"、"tea plantation"、"rice terrace" 等关键词，免费商用。

#### 2.3 能力模块图标升级
将单字（政/经/品/实/营/数）替换为精美SVG图标或Emoji组合：
- 政策解读 → 📋 或 SVG政策图标
- 经济发展 → 📈 或 SVG增长图标
- 品牌建设 → 🏷️ 或 SVG品牌图标
- 实践技能 → 🔧 或 SVG工具图标
- 营销推广 → 📢 或 SVG营销图标
- 数据分析 → 📊 或 SVG数据图标

#### 2.4 色彩体系升级
从单一蓝紫色扩展为多色系：

```
首页Hero:     蓝→紫渐变（保留）
能力模块:     每个模块独立主题色（已有，加强）
案例卡片:     根据地区用不同暖色调
课程卡片:     根据类型用不同冷色调
AI助手:       深色主题（暗色背景+霓虹高亮）
个人中心:     温暖的橙色调
```

### ⭐ P1 — 交互体验（重要）

#### 2.5 页面转场动画升级
当前：简单的 fadeIn + translateY
升级为：
- 首页→子页面：卡片展开动画（点击的卡片放大为全屏）
- 返回：反向收缩动画
- 底部导航切换：左右滑动过渡（根据tab位置方向）

#### 2.6 滚动叙事（Scroll Storytelling）
首页改为滚动驱动的叙事结构：
1. 第一屏：Hero学习卡（全屏高度，视差背景）
2. 滚动后：能力模块从底部依次浮入（stagger动画，每个延迟100ms）
3. 继续滚动：今日任务从左侧滑入
4. 最后：案例卡片以瀑布流方式逐个出现

#### 2.7 微交互增强
- 能力模块hover：圆圈图标旋转 + 背景色微变 + 进度条动画填充
- 案例卡片hover：图片微放大(scale 1.05) + 标题上移显示摘要
- 课程卡片hover：左侧缩略图轻微倾斜(rotate 2deg)
- 按钮点击：ripple水波纹效果
- 发送消息：按钮缩小弹回动画

#### 2.8 骨架屏加载
页面切换时先显示骨架屏（灰色闪烁占位），200ms后替换为真实内容，避免空白闪烁。

### 💎 P2 — 细节打磨（锦上添花）

#### 2.9 AI助手页面重设计
- 改为深色主题（#0a0a0a背景），营造科技感
- AI头像加呼吸灯光效动画
- 消息气泡加打字机效果（逐字显示）
- 快捷按钮改为发光胶囊样式

#### 2.10 空状态设计
为以下场景设计友好的空状态：
- 无收藏内容 → 插画 + "去发现好内容" 按钮
- 无学习记录 → 插画 + "开始第一课" 按钮
- 无消息通知 → 插画 + "一切安好" 文案
- 社区无消息 → 插画 + "打个招呼吧" 引导

#### 2.11 数字动画
- 首页学习天数：页面加载时从0滚动到162的计数动画
- 个人中心统计数字：同样的滚动计数效果
- 进度条：从0%动画填充到目标值

#### 2.12 底部导航升级
- 当前选中项：图标上方加发光点 + 图标微放大
- 切换时：图标有弹跳动画
- 中间的AI助手按钮可以做成凸起的圆形特殊样式
